InsticatorSession Sandbox

To observe the behavior of InsticatorSession, open the console in devtools. Call `InsticatorSession.getSession()` to create and update sessions.

(NOTE: Select 'preserve log' to observe behavior between refreshes)

  • Try changing the utm_campaign value on the search params and refreshing.

  • Try creating a session and calling it again after a short delay to see the updated expiration time.

  • Try creating a session and calling it again after a long delay to see the replacement session.

  • Try creating a session and closing and reopening the browser to see if the session is persisted.

  • And so on...
